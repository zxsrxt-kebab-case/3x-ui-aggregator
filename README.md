# 3x-ui panel subscriptions aggregator

allows to merge multiple vpn servers powered with 3x-ui panel into single one

its first version, so it will improves over time <3

## usage

add your servers into client.cfg(template included)

install golang(google how to install)

and build it: go build -o 3x-ui-aggregator

run like sudo ./3x-ui-aggregator or sudo nohup ./3x-ui-aggregator > server.log 2>&1 &
